商品是否被消费者购买
SELECT prices_isSale,CONCAT(ROUND(COUNT(prices_isSale)/(SELECT COUNT(prices_isSale) FROM DataSet)*100,2),'%') AS '占比'
FROM DataSet
GROUP BY prices_isSale;

统计数据中仅有20%商品售出，接下来进一步分析售出的鞋子属于什么品牌
SELECT brand,COUNT(brand) AS '计数'
FROM DataSet
WHERE prices_isSale='TRUE'
GROUP BY brand
ORDER BY COUNT(brand) DESC;


每个不同品牌的平均价格是多少？
选取价格排名前200的不同女鞋品牌的平均值
SELECT brand,ROUND(COUNT(*) * (prices_amountMax+prices_amountMin) / COUNT(*)*2,2) AS avgprise
FROM DataSet
GROUP BY brand
ORDER BY avgprise DESC
LIMIT 200;

哪些品牌的价格最高？
假设超过1000元为高价格品牌

SELECT brand,prices_amountMax
FROM DataSet
WHERE prices_amountMax>=1000
GROUP BY brand
ORDER BY prices_amountMax DESC;

哪些价格分布最广？
通过大致浏览价格水平，大致确认价格区间，代码如下

SELECT (CASE
WHEN prices_amountMax >0 AND prices_amountMax<= 250 THEN '0~250'
WHEN prices_amountMax >250 AND prices_amountMax<= 500 THEN '250~500'
ELSE '500以上' END) AS '价格范围',
COUNT(*) AS '数量' FROM DataSet
GROUP BY (CASE
WHEN prices_amountMax >0 AND prices_amountMax<= 250 THEN '0~250'
WHEN prices_amountMax >250 AND prices_amountMax<= 500 THEN '250~500'
ELSE '500以上' END);

通过价格数量的比较，女鞋的价格大部分集中在0-250元之间，我们在对250元以下的女鞋再进行细分
